# Steffano Continuation Suite — Trend Continuations (v1.1)

A Pine Script v6 indicator for TradingView combining:
- EMA trend alignment (20/50/200)
- ATR-based ProPlus valuation bands
- Auto Fibonacci retracements (0.382 / 0.5 / 0.618)
- Fair Value Gap (FVG) detection
- Confluence signals (EMA + Fib + FVG + ATR)
- “Unicorn” entries near 200 EMA
- Multi-timeframe scale-out alerts

## Usage
1. Copy the code in `Steffano_Continuation_Suite_v1_1.pine`
2. Paste into a new Pine Script in TradingView.
3. Add to chart → Customize settings as desired.
